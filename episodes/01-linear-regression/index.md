# 第 01 期：线性回归与温度-高程关系

## 本期问题

许多地学变量会沿着空间位置、地形和环境梯度发生变化。温度与高程的关系是理解线性回归的经典入口：在山地、青藏高原、垂直自然带、冰川和冻土研究中，我们经常需要讨论温度如何随高程变化。

气象学中有 lapse rate 的概念，用来描述温度随高度变化的速率。美国国家气象局术语表将 environmental lapse rate 定义为空气温度随高度降低的速率，通常通过探空资料测量。本期不估计真实区域的 lapse rate，而是用教学构造数据解释线性回归的基本逻辑。

核心问题是：

```text
如果我们有一组地点的高程和温度，能否用一条直线描述“高程变化时，温度平均如何变化”？
```

## 算法直觉

线性回归试图用一条直线概括两个变量之间的平均关系。对本期例子来说，横轴是高程，纵轴是温度；如果散点整体从左上往右下排列，一条向下倾斜的直线就可以表达“高程越高，温度平均越低”的趋势。

这条直线不是穿过所有点。它要做的是让观测值与预测值之间的误差整体尽可能小。scikit-learn 的 `LinearRegression` 文档将其描述为 ordinary least squares linear regression，即拟合一个线性模型，使观测目标值与线性近似预测值之间的残差平方和最小。

## 核心公式

一元线性回归的基本形式为：

```text
y = a + b x + error
```

在本期例子中：

- `y` 表示温度。
- `x` 表示高程。
- `a` 表示截距。
- `b` 表示斜率，即高程每增加一个单位时，温度的平均变化量。
- `error` 表示模型未解释的误差。

拟合后，模型会得到一条预测线：

```text
predicted_temperature = intercept + slope * elevation
```

如果斜率为负，说明在这组数据中，高程越高，模型预测的温度越低。

## 地学解释

在线性回归中，斜率是最容易被地学读者解释的参数。对于温度-高程关系，负斜率表示模型认为高程增加时温度平均降低。

但这个解释必须带着限制条件阅读：

- 这是样本中的平均关系，不代表所有地点和所有季节。
- 它描述统计关联，不自动证明完整物理机制。
- 真实温度还可能受纬度、坡向、下垫面、季节和观测时间影响。
- 如果残差有空间结构，说明模型可能遗漏了重要地学过程。

因此，线性回归适合作为理解变量关系的第一步，而不是复杂地学过程的终点。

## 适用边界

线性回归适合：

- 目标变量是连续值。
- 解释变量与目标变量之间近似线性。
- 研究目标是理解平均关系或建立基线模型。
- 样本范围足够覆盖需要解释的变化。

需要谨慎的情况包括：

- 关系明显弯曲或分段。
- 残差有强空间自相关。
- 样本点集中在少数区域。
- 异常值对斜率影响很大。
- 研究问题需要因果判断，但数据和设计只支持相关性分析。

线性回归不是过时工具，也不是万能工具。它是一种基本训练：让我们先把变量、关系、误差和边界说清楚。
