# 地学人工智能算法①：回归分析到底能解决什么问题？

## 本期问题

很多人一提到“算法”，会先想到随机森林、神经网络和大模型。但在地学研究中，回归分析仍然是最常用、最经典、也最有生命力的方法之一。

原因很直接：地学研究里有大量问题，本质上都在追问变量之间的关系。

- 地表温度是否会随高程升高而降低？
- 湖泊面积变化能否由降水和温度解释？
- NDVI 的空间差异是否与降水、温度、土壤湿度或高程有关？
- 冻土活动层厚度是否受到气候和地形因子的共同控制？
- 某区域滑坡发生概率是否与坡度、坡向、降雨和人类活动强度相关？

只要研究目标涉及影响因素、驱动机制、变量关系或预测趋势，就很可能会用到回归分析。

本期使用 `NDVI = Precip + Temp + Elevation` 作为教学案例，说明回归分析如何帮助地学研究者描述关系、建立预测、比较变量影响，并诊断模型问题。示例数据由 notebook 构造，不代表真实观测。

## 什么是回归分析

回归分析的核心目标是：

```text
用一个数学模型描述因变量如何随着自变量变化。
```

因变量是我们关心的结果，例如 NDVI、湖泊面积、地表温度、土壤含水量或污染物浓度。自变量是可能影响结果的因素，例如降水、温度、高程、坡度、土地覆盖或人类活动强度。

最简单的一元线性回归可以写成：

```text
y = a + b x + error
```

其中，`y` 是因变量，`x` 是自变量，`a` 是截距，`b` 是回归系数，`error` 是模型没有解释掉的部分。

如果有多个影响因素，就变成多元线性回归：

```text
y = a + b1 x1 + b2 x2 + b3 x3 + ... + bn xn + error
```

本期例子对应：

```text
NDVI = a + b1 Precip + b2 Temp + b3 Elevation + error
```

scikit-learn 的 `LinearRegression` 文档将该方法描述为 ordinary least squares linear regression。也就是说，它会拟合一个线性模型，使观测值与线性预测值之间的残差平方和尽量小。

## 回归分析在地学里能做什么

### 1. 解释关系

回归可以把“看起来有关”转化为“定量地说明有多大关系”。例如，当我们研究湖泊面积变化时，可以检验降水、气温和蒸发等变量是否与湖泊面积存在统计关系。

这类解释不等于因果证明。它更准确的含义是：在当前数据和模型设定下，变量之间呈现怎样的统计关联。

### 2. 预测结果

回归也可以用于预测。例如，根据气候因子预测 NDVI，根据地形和气温预测地表温度，根据环境变量预测土壤含水量。

它未必总是预测能力最强的模型，但结果透明、结构清楚、便于解释，适合作为复杂模型之前的基线。

### 3. 识别驱动因素

地学研究常常不只追求“预测准”，还关心为什么会变化、哪个因素更重要、变量影响方向是否符合地学认识。

回归系数、标准化系数、显著性检验、残差和共线性诊断，为这些问题提供了可检查的入口。

## 地学中常见的几类回归

### 一元线性回归

一元线性回归只有一个自变量，适合做初步探索。

```text
NDVI = a + b Precip + error
```

例如，高程与地表温度、降水与 NDVI、时间与湖泊面积，都可以先用一元回归建立直觉。

![降水与 NDVI 的教学构造数据散点图](figures/precip_ndvi_scatter.png)

### 多元线性回归

多元线性回归同时纳入多个自变量，是地学中非常常见的建模形式。

```text
NDVI = a + b1 Precip + b2 Temp + b3 Elevation + error
```

它的价值在于把多个地学因素放到同一个模型中比较。例如，在控制气温和高程之后，降水是否仍然与 NDVI 相关？在控制降水和气温之后，高程是否仍然具有解释力？

### 非线性回归

许多地学过程并不是直线关系。植被对温度可能存在适宜范围，水体面积对降水可能存在阈值，冻土热过程也可能呈现非线性响应。

这时可以考虑二次项、对数项、指数项、分段函数或其他非线性形式。但复杂项应来自机制假设、图形证据或明确研究问题，而不是为了提高拟合度随意添加。

### 逻辑回归

逻辑回归虽然叫“回归”，但常用于二分类和概率建模。例如：

- 某位置是否发生滑坡。
- 某像元是否为水体。
- 某区域是否属于高风险区。

残差是观测值减去预测值。理想情况下，残差应围绕 0 随机分布。如果残差呈现弯曲、扇形扩散或结构性分布，说明模型可能遗漏了非线性关系、异方差或关键变量。

![方差膨胀因子 VIF 图](figures/vif_plot.png)

VIF 用于检查解释变量之间的共线性。statsmodels 的 `variance_inflation_factor` 文档指出，如果某变量的 VIF 大于 5，说明该变量与其他解释变量高度共线。共线性不一定破坏预测，但会削弱单个系数的稳定解释。

## 地学解释

回归分析的关键不只是得到一个预测值，而是把模型结果翻译回地学问题。

如果降水系数为正，可以解释为：在当前模型和样本范围内，降水较高的位置通常对应较高 NDVI。如果高程系数为负，可以解释为：在控制其他变量之后，高程较高的位置 NDVI 较低。

但这些解释必须加上限定条件：

- 这是统计关联，不是自动因果结论。
- 结论只适用于当前样本范围。
- 如果变量之间高度相关，单个系数的解释会不稳定。
- 如果残差具有空间结构，普通回归可能低估了空间过程的重要性。

## 适用边界

回归分析适合：

- 描述变量关系。
- 建立可解释预测模型。
- 比较多个驱动因素。
- 作为复杂 AI 模型之前的基线。
- 发现残差、异常点和潜在遗漏变量。

但它不能：

- 把相关关系直接变成因果关系。
- 替代严谨的数据设计和采样方案。
- 自动处理空间自相关。
- 在数据支持范围之外无限外推。
- 用一个线性模型解释所有复杂地学过程。

因此，回归分析不是过时工具，也不是万能工具。它是一种基本训练：让我们先把问题、变量、关系、误差和边界说清楚。
