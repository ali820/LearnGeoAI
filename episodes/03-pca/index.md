# 第 03 期：主成分分析与多源环境变量降维

## 本期问题

地学研究经常同时使用多源环境变量，例如气候、地形、植被、水文和人类活动因子。变量数量增加后，研究者会遇到一个常见问题：

```text
当多个环境变量彼此相关时，能否用更少的综合变量概括主要信息？
```

主成分分析（Principal Component Analysis, PCA）是一种经典多变量统计方法，在机器学习中常作为无监督降维和特征提取工具使用。本期用多源环境变量降维作为案例，讲解 PCA 的基本直觉、计算流程、图件解释和地学应用边界。

本期示例使用教学构造数据，不代表真实区域观测结果。

## 算法直觉

PCA 的目标是在尽量保留原始数据主要方差信息的前提下，把一组相关变量转换成少数几个互不相关的综合变量，即主成分。

可以把 PCA 理解为一次坐标轴旋转：

1. 原始坐标轴是降水、温度、高程、坡度、NDVI 等变量。
2. PCA 寻找新的坐标轴，使样本在第一个轴上的方差最大。
3. 第二个轴与第一个轴正交，并解释剩余方差中尽可能多的信息。
4. 后续主成分依次类推。

主成分不是原始地学因子，而是由原始变量线性组合得到的新变量。解释主成分时，需要结合载荷、研究区域背景和地学知识。

## 核心公式

设标准化后的数据矩阵为 `X`，PCA 将原始变量转换为新的主成分得分：

```text
Z = XW
```

其中：

- `X`：标准化后的样本-变量矩阵
- `W`：主成分方向矩阵
- `Z`：样本在主成分空间中的得分

第 k 个主成分可以写成：

```text
PC_k = w_1k x_1 + w_2k x_2 + ... + w_pk x_p
```

其中 `w_ik` 是第 i 个原始变量在第 k 个主成分上的权重或载荷。

PCA 通常关注两个结果：

- **解释方差比例**：每个主成分保留的原始数据方差信息比例。
- **主成分载荷**：原始变量与主成分之间的关系方向和相对强度。

## 地学解释

在本期演示数据中，变量被设计为包含水分梯度、地形梯度和温度梯度等共同结构。因此，PCA 结果可以用于说明主成分如何概括相关变量的信息。

解释时应注意：

- PC1、PC2 是数学构造的新变量，不是直接观测到的地学过程。
- 载荷可以帮助命名主成分，但命名本身属于研究者解释。
- 解释方差高不等于对某个目标变量预测能力强。
- 若要把 PCA 结果用于回归、分类或聚类，需要在后续任务中单独验证效果。
- 若 PCA 被用于训练-测试预测流程，标准化和 PCA 应只在训练集上拟合，再应用到验证集或测试集，避免数据泄漏。

## 适用边界

PCA 适合：

- 多个连续变量之间存在相关性。
- 需要减少变量冗余或降低维度。
- 需要把高维样本投影到二维或三维空间中进行探索。
- 后续模型希望使用互不相关的综合特征。

PCA 不适合直接替代：

- 具体变量的机制解释。
- 非线性结构很强的降维任务。
- 对异常值和缺失值未经处理的数据分析。
- 有明确目标变量且需要直接优化预测性能的任务。
- 多源空间变量尚未完成投影、分辨率、范围、掩膜和时间窗口对齐的分析。
