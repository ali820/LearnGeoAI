# 第 03 期示例数据说明

## 文件

- `environment_pca_demo.csv`

## 数据性质

该数据由 `episodes/03-pca/notebook.ipynb` 生成，是用于讲解 PCA 的教学构造数据，不代表真实区域观测结果。

## 字段

| 字段 | 含义 | 单位或范围 |
| --- | --- | --- |
| `sample_id` | 样本编号 | 无 |
| `terrain_group` | 教学分组标签，仅用于绘图着色 | lowland / hillslope / upland |
| `precip` | 年降水量 | mm |
| `temp` | 年均温 | 摄氏度 |
| `elevation` | 高程 | m |
| `slope` | 坡度 | degree |
| `ndvi` | 归一化植被指数 | 0-1 |
| `soil_moisture` | 土壤湿度指数 | 0-1 |
| `aridity` | 干旱指数 | 无量纲 |

## 生成逻辑

示例数据通过三个潜在梯度构造：

- 水分梯度：影响降水、土壤湿度、NDVI 和干旱指数。
- 地形梯度：影响高程、坡度和部分气温变化。
- 温度梯度：影响气温、NDVI 和干旱指数。

生成过程中加入随机扰动，以模拟变量之间相关但不完全重复的情况。

## 使用限制

该数据仅用于方法演示和代码复现，不应用于真实地学结论、模型性能比较或工程决策。
