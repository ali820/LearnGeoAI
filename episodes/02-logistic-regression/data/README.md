## 数据预处理注意事项

1. **坐标系统一**：确保所有数据使用相同坐标系
2. **分辨率匹配**：重采样到统一分辨率
3. **缺失值处理**：插值或剔除
4. **样本点提取**：从栅格中提取滑坡点和非滑坡点的特征值
5. **非滑坡点采样**：可采用随机采样或匹配采样策略
